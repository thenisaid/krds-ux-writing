## 🟡 F2 — 에러 메시지 구조 라벨 차이 (MEDIUM)

**현황**: "② 사용자 잘못이 아님을 암시" 표기 (P2 규칙3 / P3 B/A② / P4 실습② 공통)

**principles.md 4.1 공식 3단 구조**:
```
1단 — 상황(What): 무슨 일이 있었는지
2단 — 원인(Why): 시스템 측 원인 우선, 사용자 탓 금지
3단 — 행동(How): 지금 무엇을 하면 되는지
```

수정:
- ❌ `② 사용자 잘못이 아님을 암시`
- ✓ `② 왜 그런지 (시스템 원인 우선, 사용자 탓 금지)`
